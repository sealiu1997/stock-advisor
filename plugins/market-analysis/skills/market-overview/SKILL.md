## Step 3: 抓取全球指数

使用 yfinance 批量下载全球主要指数最近 2 天数据：

```python
import yfinance as yf
import pandas as pd

indices = {
    # 美股
    "^GSPC": "标普500",
    "^IXIC": "纳斯达克",
    "^DJI": "道琼斯",
    "^RUT": "罗素2000",
    # 亚太
    "^HSI": "恒生指数",
    "000001.SS": "上证综指",
    "^N225": "日经225",
    # 欧洲
    "^STOXX50E": "欧洲STOXX50",
}

data = yf.download(list(indices.keys()), period="2d", progress=False)
```

**计算涨跌幅：** `(最新收盘 / 前一日收盘 - 1) * 100`

**输出格式：**

```
### 全球主要指数
| 指数 | 点位 | 涨跌幅 | 方向 |
|------|------|--------|------|
| 标普500 | 5,432.10 | +0.85% | ↑ |
| 纳斯达克 | 17,123.45 | +1.20% | ↑ |
...
```

## Step 4: 抓取宏观风向标

这些指标帮助判断市场整体风险偏好：

```python
macro_symbols = {
    "^VIX": "VIX恐慌指数",
    "^TNX": "10Y美债收益率",
    "^FVX": "5Y美债收益率",
    "DX-Y.NYB": "美元指数",
    "GC=F": "黄金期货",
    "CL=F": "原油期货",
}

macro_data = yf.download(list(macro_symbols.keys()), period="5d", progress=False)
```

**解读规则（写入输出的"信号"列）：**

| 指标 | 解读 |
|---|---|
| VIX < 15 | 市场极度乐观（贪婪） |
| VIX 15-20 | 市场平稳 |
| VIX 20-30 | 市场焦虑 |
| VIX > 30 | 市场恐慌 |
| 10Y 收益率上升 | 利率预期上行，成长股承压 |
| 10Y 收益率下降 | 降息预期，利好成长股 |
| 美元指数上升 | 资金回流美元，新兴市场承压 |
| 美元指数下降 | 风险偏好上升，利好商品和新兴市场 |
| 黄金上涨 + VIX上升 | 避险情绪主导 |
| 原油上涨 | 通胀预期或供应紧张 |

**输出格式：**

```
### 宏观风向标
| 指标 | 当前值 | 变动 | 信号 |
|------|--------|------|------|
| VIX 恐慌指数 | 18.5 | -1.2 (-6.1%) | 市场平稳 |
| 10Y 美债 | 4.25% | +3bp | 利率微升 |
| 美元指数 | 104.3 | +0.2% | 美元偏强 |
| 黄金 | $2,345 | +0.5% | 避险需求温和 |
| 原油 WTI | $78.5 | -1.2% | 需求预期偏弱 |
```

## Step 5: 板块热力图

使用美股 11 大 GICS 板块 ETF 衡量板块轮动：

```python
sector_etfs = {
    "XLK": "科技",
    "XLF": "金融",
    "XLE": "能源",
    "XLV": "医疗",
    "XLY": "可选消费",
    "XLP": "必需消费",
    "XLI": "工业",
    "XLB": "材料",
    "XLRE": "房地产",
    "XLU": "公用事业",
    "XLC": "通信",
}

sector_data = yf.download(list(sector_etfs.keys()), period="2d", progress=False)
```

**输出格式（按涨跌幅降序排列）：**

```
### 板块热力图 (GICS 11大板块)
| 板块 | ETF | 涨跌幅 | 强弱 |
|------|-----|--------|------|
| 科技 | XLK | +1.8% | 领涨 |
| 通信 | XLC | +1.2% | 强势 |
| 金融 | XLF | +0.3% | 中性 |
...
| 公用事业 | XLU | -0.5% | 落后 |
```

**强弱标注规则：**
- 涨幅 > 1.5%: 领涨
- 涨幅 0.5% - 1.5%: 强势
- 涨幅 -0.5% - 0.5%: 中性
- 跌幅 0.5% - 1.5%: 偏弱
- 跌幅 > 1.5%: 领跌

**板块轮动解读：**
- 科技 + 可选消费领涨 → Risk-On（风险偏好上升）
- 公用事业 + 必需消费领涨 → Risk-Off（避险情绪）
- 能源领涨 → 通胀/供应主题
- 金融领涨 → 利率上行预期

## Step 6: 市场情绪综合评估

基于以上数据，给出综合评估。这是 agent 最核心的分析输出。

### 评估框架

**1. 恐惧-贪婪维度**

综合以下信号给出 1-100 的情绪评分：
- VIX 水平和变化方向
- 指数涨跌面（涨多跌少？）
- 板块分化程度（领涨板块 vs 领跌板块的差距）
- 避险资产表现（黄金、美债、美元）

| 评分 | 状态 |
|---|---|
| 0-25 | 极度恐惧 |
| 25-45 | 恐惧 |
| 45-55 | 中性 |
| 55-75 | 贪婪 |
| 75-100 | 极度贪婪 |

**2. Risk-On / Risk-Off 判定**

| 信号组合 | 判定 |
|---|---|
| 成长板块领涨 + VIX下降 + 美债收益率上升 | Risk-On |
| 防御板块领涨 + VIX上升 + 黄金上涨 | Risk-Off |
| 信号混杂 | 混合 / 观望 |

**3. 主要驱动因素**

识别 1-3 个当前市场的核心驱动因素（如："AI板块持续走强"、"降息预期升温"、"地缘风险上升"）

**4. 短期展望**

结合以上分析，给出未来 1-3 个交易日的方向性展望和关注要点。

### 输出格式

```
### 市场情绪综合评估

**情绪评分：62/100 (贪婪)**
**风险偏好：Risk-On**

**驱动因素：**
1. AI/科技板块持续领涨，NVDA 带动半导体上行
2. CPI 数据低于预期，降息预期升温

**短期展望：**
市场整体偏多，科技板块动能充沛。关注点：...
```

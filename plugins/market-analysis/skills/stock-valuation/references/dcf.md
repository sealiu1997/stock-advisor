# DCF 方法论详细参考

## 适用场景

**适合 DCF：** 成熟公司、可预测现金流、战略性估值、交叉验证相对估值
**不适合 DCF：** 未盈利/早期公司、银行保险（用 DDM）、REITs（用 NAV）、强周期性无基线

## 投影模型 (5 年显式预测)

### 营收投影

1. 计算历史 3-5 年 CAGR
2. 获取分析师共识（yfinance `revenue_estimate`）
3. 考虑行业增长、竞争地位、公司指引
4. 从 Y1 到 Y5 线性衰减至终端增长率

### FCFF 构建

```
营收
- 销售成本 (COGS)              → 历史毛利率趋势
= 毛利
- 销售与管理费用 (SG&A)        → 历史占营收比
- 研发费用 (R&D)               → 历史占营收比
= 营业利润 (EBIT)

FCFF = EBIT × (1 − 税率)
     + 折旧摊销 (D&A)
     + 股权激励 (SBC)           ← 仅当视为非现金项时
     − 资本支出 (CapEx)
     − 净营运资本变动 (ΔNWC)
```

### 假设清单

| 假设 | 推导方式 | 典型范围 |
|---|---|---|
| 税率 | 历史有效税率 | 美国 15-25%；不可靠时用法定税率 |
| D&A | 占营收% 或 PP&E 折旧表 | 多数 3-8%；电信/公用 15-25% |
| CapEx | 占营收%；区分维护性/成长性 | SaaS 3-8%；工业 8-15%；电信 15-25% |
| NWC 变动 | DSO/DPO/DIO | 通常为营收变动的 1-3% |
| SBC | 软件/SaaS 视为现金；工业/消费视为非现金 | 提前决定并披露 |

## WACC 计算

```
WACC = (E/V) × Ke + (D/V) × Kd × (1 − 税率)
Ke = 无风险利率 + Beta × 股权风险溢价 + 规模溢价（如适用）
```

**资本结构用市场价值：** E = 市值, D = 总债务, V = E + D

## 终值

### 永续增长法 (Gordon Growth)
```
TV = FCFF_5 × (1 + g) / (WACC − g)
```
- 终端增长 g: 美国 2-3%，不应超过长期名义 GDP
- TV 通常占 EV 的 60-80%，超出此范围需标记

### 退出倍数法
```
TV = EBITDA_5 × 退出 EV/EBITDA 倍数
```
- 用当前同业交易倍数作参考
- 考虑 Y5 增长减速的折价
- 两种方法偏差 >30% 需调和假设

## 企业价值桥接至股权

```
PV(FCFF) = Σ FCFF_t / (1 + WACC)^t
PV(TV)   = TV / (1 + WACC)^5
企业价值 = PV(FCFF) + PV(TV)
+ 现金及等价物
− 总债务
− 少数股东权益
− 优先股
+ 股权投资（如重大）
= 股权价值
隐含股价 = 股权价值 / 稀释后流通股
```

## 行业特定指南

### 科技 / SaaS
- EV/Revenue 常比 P/E 更有意义（可能尚未盈利）
- 关键指标：ARR 增长、净收入留存率 (NRR)、Rule of 40
- CapEx 轻（3-8%）；R&D 重（15-30%）
- SBC 重大 — 提前决定处理方式
- 终端增长：品类领导者 3-4%，其他 2-3%

### 零售 / 电商
- 营收 = 同店增长 + 新店扩张（实体）或 GMV 增长（线上）
- 营运资本重要：库存周转、应付账款
- CapEx 区分维护性（现有门店）和成长性（新门店/仓储）

### 金融服务（银行/保险）
- 标准 DCF 不适用。用 DDM 或超额收益模型
- 如被迫使用：分别预测 NII、拨备、非利息收入
- 折现率 = 仅用股权成本（债务是经营性的）

### 医疗 / 制药
- 分开现有产品组合和研发管线
- 关键风险：专利悬崖、FDA 审批概率
- 生物科技：按临床阶段成功概率对管线 NPV 风险调整

### 能源 (油气)
- 营收与商品价格挂钩 — 用远期曲线或情景分析
- CapEx 高；区分开发与勘探
- 终值对长期油价假设极度敏感

### 制造业 / 工业
- 周期性 — 用中周期盈利归一化
- CapEx 8-15%
- 营运资本随周期波动 — 用穿越周期均值

### 消费品 (CPG)
- 稳定、可预测 — DCF 好标的
- 区分有机增长和并购增长
- 关注毛利率趋势、品牌投入、原材料成本

### 电信
- CapEx 高（15-25%）用于网络建设
- 经常性收入、低流失 — 适合 DCF
- WACC 7-9%（大型运营商）

## 常见陷阱

- **终值主导**：TV > 80% EV 意味着实质上是倍数扩张赌注
- **增长率 > WACC**：Gordon 公式无效，必须封顶
- **SBC 双重计算**：要么从 FCFF 扣除 SBC，要么用含 SBC 稀释的股数 — 不能同时不做或同时做
- **忽略少数股东权益/优先股**：这些是优先于普通股的 EV 索求权
- **循环 WACC**：WACC 用市值计算，而市值正是我们要估的。对 IPO 或争议标的需迭代

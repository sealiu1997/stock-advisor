# 相对估值详细参考

## 同业选择规则

目标 4-6 家，多了噪音大，少了脆弱：

| 标准 | 优先级 |
|---|---|
| 同 GICS 行业 | 必须 |
| 类似商业模式 | 必须 |
| 类似增长率 (±10 ppt) | 强偏好 |
| 类似利润率 | 偏好 |
| 类似资本结构 | 加分 |
| 类似地域敞口 | 加分 |

**避免：** 超大盘多元化公司作为纯业务中小盘同业（如 MSFT 不适合做 DDOG 的同业）

## 倍数速查表

| 倍数 | 适合 | 不适合 |
|---|---|---|
| P/E (滚动) | 成熟盈利公司 | 亏损、周期谷底 |
| P/E (远期) | 成长、盈利可见 | 早期、预测分歧大 |
| PEG (P/E ÷ 增长) | 高成长盈利 | 成熟低增长 |
| EV/Revenue | 未盈利、早期 SaaS | 成熟混合利润率 |
| EV/EBITDA | 中后期、跨资本结构 | 金融、REITs |
| EV/EBIT | 资本密集型 | D&A 会计不一致 |
| P/B | 银行、保险 | 轻资产公司 |
| P/FFO, P/AFFO | REITs | 其他 |

## 计算隐含价格

取同业**中位数**（非均值，抗异常值）：

```
# 股权倍数
隐含价格 (P/E) = 同业中位数 P/E × 目标 EPS_TTM

# 企业价值倍数
隐含 EV = 同业中位数 EV/Rev × 目标 Revenue_TTM
净债务 = 总债务 − 现金
隐含股权价值 = 隐含 EV − 净债务 − 少数股东权益 − 优先股
隐含价格 = 隐含股权价值 / 稀释后股数
```

## 调整规则

当目标公司与同业中位数差异显著时，调整隐含倍数 ±10-30%：

| 目标公司具有... | 调整隐含倍数 |
|---|---|
| 更高增长率 (>5ppt 以上) | +10% 到 +30% |
| 更低增长率 | −10% 到 −30% |
| 更高利润率 (>3ppt 以上) | +10% 到 +20% |
| 更低利润率 | −10% 到 −20% |
| 更好资产负债表/更低杠杆 | +5% 到 +10% |
| 更高杠杆/契约风险 | −10% 到 −20% |
| 主导市场地位/护城河 | +10% 到 +20% |
| 市场份额流失 | −10% 到 −20% |
| 监管悬顶/维权投资者 | −5% 到 −15% |

**始终说明调整及原因。**

## Rule of 40（SaaS 专用）

```
Rule of 40 = 营收增长% + FCF 利润率%
```

| Rule of 40 评分 | 同业 EV/Revenue 溢价 |
|---|---|
| ≥ 50 | 顶部四分位 — 用 75 分位同业倍数 |
| 40-50 | 高于中位 — 中位数 + 10% |
| 30-40 | 低于中位 — 中位数 − 10% |
| < 30 | 底部四分位 — 用 25 分位同业倍数 |

## 常见陷阱

- **单一倍数**：至少用 ≥2 个倍数交叉验证
- **异常值同业**：P/E > 100 或 EV/Rev > 50 需排除（除非目标也是极端值）
- **同业处于低谷**：困境或重组中的同业倍数受压，需排除或调整
- **不同财年结束日**：统一为 TTM
- **SBC 未调整**：SaaS 同业未扣 SBC 的 EV/EBITDA 会虚高
- **货币差异**：国际同业需统一为美元并注明汇率敏感性

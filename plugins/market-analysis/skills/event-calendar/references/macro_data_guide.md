# 宏观数据指标解读指南

## CPI (Consumer Price Index / 消费者物价指数)

**含义：** 衡量消费者购买的一篮子商品和服务的价格变化，是通胀的核心指标。

**关键数据点：**
- **CPI YoY (同比)**：最受关注，反映整体通胀趋势
- **CPI MoM (环比)**：月度变化，反映短期通胀动能
- **核心 CPI (Core CPI)**：剔除食品和能源的波动，反映"粘性"通胀

**市场影响机制：**

| CPI 表现 | 市场反应 | 逻辑 |
|---|---|---|
| 大幅低于预期 | 股市涨，债券涨，美元跌 | 降息预期升温 |
| 小幅低于预期 | 温和利好 | 通胀受控信号 |
| 符合预期 | 影响有限 | 已被定价 |
| 高于预期 | 股市跌，债券跌，美元涨 | 加息/不降息预期 |

**敏感资产：**
- 高度敏感：成长股 (QQQ)、房地产 (XLRE)、长期国债 (TLT)
- 中度敏感：标普500、黄金
- 低敏感：能源、必需消费

## Non-Farm Payrolls (非农就业)

**含义：** 美国非农业部门当月新增就业人数，是美联储"双重使命"(就业+通胀) 的直接指标。

**关键数据点：**
- **新增就业人数**：核心数字，市场预期通常在 15-25 万
- **失业率**：百分比，当前处于历史低位附近
- **平均时薪 (Average Hourly Earnings)**：工资通胀，美联储高度关注

**市场影响机制：**

| 非农表现 | 市场反应 | 逻辑 |
|---|---|---|
| 远超预期 (如 +30万) | 好坏参半 | 经济强劲但降息推迟 |
| 略超预期 | 温和利好 | 经济健康 + 政策可控 |
| 符合预期 | 影响有限 | |
| 不及预期 (如 <10万) | 先跌后可能涨 | 衰退担忧 vs 降息预期 |
| 时薪超预期 | 股市承压 | 工资通胀 → 利率维持高位 |

**"坏消息即好消息"现象：** 在美联储加息/维持高利率周期中，偏弱的就业数据有时反而利好股市，因为市场预期美联储会更快降息。

## FOMC Rate Decision (美联储利率决议)

**含义：** 美联储设定联邦基金利率目标区间，影响全球资金成本。

**关注内容优先级：**
1. **利率决定本身**：加息/降息/维持
2. **声明措辞变化**：与上一次对比，关键词变化
3. **点阵图 (SEP 会议)**：委员对未来利率路径的预测
4. **鲍威尔记者会**：措辞和语气比声明更灵活

**关键措辞信号：**

| 措辞 | 含义 |
|---|---|
| "further tightening" | 可能继续加息 |
| "maintain restrictive stance" | 暂停加息但不急于降 |
| "prepared to adjust" | 态度转向灵活 |
| "balanced risks" | 偏鸽，接近降息 |
| "rate cuts appropriate" | 明确降息信号 |

## PMI (Purchasing Managers' Index / 采购经理人指数)

**含义：** 基于采购经理人调查的扩散指标，50 为荣枯线。

**版本区分：**
- **ISM 制造业 PMI**：美国制造业景气度
- **ISM 服务业 PMI**：美国服务业景气度（占 GDP ~70%）
- **S&P Global PMI**：全球版本，初值在月中发布（更早但样本小）

**阈值解读：**

| 区间 | 含义 |
|---|---|
| > 55 | 强劲扩张 |
| 50-55 | 温和扩张 |
| 48-50 | 临界区域，增长放缓 |
| 45-48 | 收缩 |
| < 45 | 深度收缩 |

**子项关注：**
- 新订单 (New Orders)：最具前瞻性
- 就业 (Employment)：预示非农方向
- 价格 (Prices Paid)：通胀前瞻指标

## GDP (Gross Domestic Product / 国内生产总值)

**含义：** 经济总量的季度变化，衡量经济增长速度。

**版本时序：**
1. **初值 (Advance)**：季度结束后约 1 个月，数据不完整但市场影响最大
2. **修正值 (Second)**：季度结束后约 2 个月
3. **终值 (Third)**：季度结束后约 3 个月，市场关注度低

**关键区间：**

| 年化增速 | 含义 |
|---|---|
| > 3% | 强劲增长 |
| 2-3% | 健康增长（美国长期趋势） |
| 1-2% | 增长放缓 |
| 0-1% | 接近停滞 |
| < 0% | 技术性衰退风险（连续两季负增长） |

---

## PCE Price Index (个人消费支出物价指数)

**含义：** 美联储最看重的通胀指标（优先于 CPI）。

**与 CPI 的区别：**
- 覆盖范围更广（包括雇主代付的医疗保险等）
- 权重动态调整（反映消费者替代行为）
- 通常低于 CPI 约 0.3-0.5 个百分点

**美联储目标：** 核心 PCE YoY = 2.0%

| 核心PCE | 政策含义 |
|---|---|
| < 2.0% | 通缩风险，可能宽松 |
| 2.0-2.5% | 目标区间，政策中性 |
| 2.5-3.0% | 偏高，维持紧缩 |
| > 3.0% | 明显过高，加息/维持高利率 |

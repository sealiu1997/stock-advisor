# 宏观指标解读指南

## VIX 恐慌指数

**含义：** 标普 500 指数未来 30 天隐含波动率，反映市场恐惧程度。

**历史分位：**

| 区间 | 解读 | 历史占比 |
|---|---|---|
| < 12 | 极度平静，可能暗示自满 | ~10% |
| 12-15 | 低波动，牛市常态 | ~25% |
| 15-20 | 正常水平 | ~35% |
| 20-25 | 焦虑上升 | ~15% |
| 25-30 | 明显恐慌 | ~8% |
| 30-40 | 高度恐慌 | ~5% |
| > 40 | 极度恐慌 (危机级别) | ~2% |

**关键阈值：**
- VIX 突破 20 → 风险偏好转向谨慎
- VIX 突破 30 → 市场可能进入恐慌抛售
- VIX 从高位快速回落 → 可能是反弹信号

## 10Y 美债收益率 (^TNX)

**含义：** 10 年期美国国债收益率，全球资产定价之锚。

**影响链条：**
- 收益率上升 → 折现率上升 → 成长股估值承压（DCF 分母变大）
- 收益率下降 → 降息预期 → 成长股和长久期资产利好
- 2Y-10Y 倒挂 → 衰退预警信号

**关键区间 (2024-2026 周期)：**

| 区间 | 市场含义 |
|---|---|
| < 3.5% | 降息已落地，利好股市 |
| 3.5-4.0% | 偏中性，市场消化降息 |
| 4.0-4.5% | 利率维持高位，成长股承压 |
| 4.5-5.0% | 加息预期重燃，市场承压 |
| > 5.0% | 高度紧缩，风险资产普跌 |

## 美元指数 (DX-Y.NYB)

**含义：** 美元相对六种主要货币的加权指数（欧元占 57.6%）。

**影响链条：**
- 美元走强 → 新兴市场货币贬值 → 资金回流美国 → 大宗商品承压
- 美元走弱 → 出口竞争力上升 → 新兴市场和商品利好
- 美元与黄金通常负相关

**关键区间：**

| 区间 | 状态 |
|---|---|
| < 100 | 美元偏弱 |
| 100-103 | 中性 |
| 103-106 | 美元偏强 |
| > 106 | 美元强势 |

## 黄金 (GC=F)

**驱动因素优先级：**
1. 实际利率（名义利率 - 通胀预期）：实际利率下降 → 黄金上涨
2. 美元强弱：美元走弱 → 黄金上涨
3. 地缘政治风险：不确定性上升 → 避险买盘
4. 央行购金：新兴市场央行去美元化需求

**信号组合解读：**
- 黄金涨 + 美债收益率降 + 美元降 → 典型降息交易
- 黄金涨 + 美元也涨 → 避险驱动（地缘风险）
- 黄金涨 + 通胀预期升 → 通胀对冲需求

## 原油 WTI (CL=F)

**驱动因素：**
1. OPEC+ 减产/增产决策
2. 全球经济增长预期（需求端）
3. 地缘政治（中东局势、俄乌冲突）
4. 美国战略储备 (SPR) 买卖
5. 季节性（夏季驾驶季需求上升）

**关键区间 (2024-2026)：**

| 区间 | 含义 |
|---|---|
| < $60 | 需求疲软或供应过剩 |
| $60-75 | 供需平衡偏松 |
| $75-90 | 正常区间 |
| $90-100 | 供应紧张或地缘溢价 |
| > $100 | 供应危机 |

## GICS 11 大板块 ETF

| ETF | 板块 | 典型特征 |
|---|---|---|
| XLK | 科技 | 成长、高估值、利率敏感 |
| XLF | 金融 | 受益利率上升、经济周期 |
| XLE | 能源 | 商品周期、通胀对冲 |
| XLV | 医疗 | 防御、弱周期 |
| XLY | 可选消费 | 经济敏感、消费信心 |
| XLP | 必需消费 | 防御、低波动 |
| XLI | 工业 | 经济周期、基建 |
| XLB | 材料 | 商品周期、通胀 |
| XLRE | 房地产 | 利率极度敏感 |
| XLU | 公用事业 | 债券替代、利率敏感(反向) |
| XLC | 通信 | 混合（META, GOOG 占比大） |

**板块轮动信号：**
- 科技+可选消费领涨 → 经济扩张期初期 / Risk-On
- 金融+工业领涨 → 经济扩张中期
- 能源+材料领涨 → 通胀上行 / 经济过热
- 公用事业+必需消费+医疗领涨 → 经济放缓 / Risk-Off
- 房地产领涨 → 降息预期强烈
